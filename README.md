# Hospital Management System
A full-stack web application to manage patients, doctors, appointments, and billing.

## Technologies Used

- HTML, CSS, JavaScript  
- Node.js, Express.js  
- MySQL  

## Features

- Login system  
- Manage patients & doctors  
- Book appointments  
- Billing system  


## Run Project
npm install  
node server.js
